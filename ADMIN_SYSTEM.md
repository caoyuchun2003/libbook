# 后台管理系统

## 功能概述

华腾教育 后台管理系统为管理员提供完整的数据管理功能，包括图书管理、用户管理、分类管理、角色管理和数据统计。

## 访问方式

### 登录

使用管理员账号登录：
- 邮箱：admin@libbook.com
- 密码：password

### 进入后台

登录后，在左侧菜单点击"后台管理"，或直接访问：`/admin`

## 功能模块

### 1. 管理首页 (`/admin`)

- **数据统计卡片**
  - 图书总数
  - 用户总数
  - 借阅总数
  - 进行中借阅

- **最近借阅记录**
  - 显示最近 5 条借阅记录
  - 包含图书、借阅人、日期、状态

- **快速操作**
  - 添加图书
  - 管理分类
  - 用户管理
  - 借阅记录

### 2. 图书管理 (`/admin/books`)

- 图书列表（支持搜索和分类筛选）
- 添加图书
- 编辑图书信息
- 删除图书
- 查看图书详情

### 3. 分类管理 (`/admin/categories`)

- 分类列表
- 添加分类
- 编辑分类
- 删除分类

### 4. 用户管理 (`/admin/users`)

- 用户列表（支持搜索）
- 编辑用户信息
- 修改用户角色（普通用户/管理员）
- 删除用户（不能删除自己，不能删除最后一个管理员）

### 5. 借阅记录 (`/admin/borrow-records`)

- 所有借阅记录列表
- 按状态筛选（借阅中/已归还/逾期）
- 搜索功能
- 归还操作

### 6. 数据统计 (`/admin/statistics`)

- 图书统计
- 借阅统计
- 用户统计
- 热门图书排行

## 权限控制

- **路由守卫**：只有管理员可以访问 `/admin/*` 路由
- **API 权限**：用户管理 API 需要管理员权限
- **自动跳转**：普通用户访问后台会自动跳转到用户端首页

## 技术实现

### 前端

- **布局**：`AdminLayout.vue` - 独立的管理后台布局
- **路由**：`meta: { requiresAdmin: true }` - 需要管理员权限
- **权限检查**：使用 `authStore.isAdmin` 判断

### 后端

- **中间件**：`AdminMiddleware` - 验证管理员权限
- **API 路由**：`Route::middleware('admin')` - 保护管理员 API

## API 端点

### 用户管理（需要管理员权限）

- `GET /api/users` - 获取用户列表
- `GET /api/users/{id}` - 获取用户详情
- `PUT /api/users/{id}` - 更新用户信息
- `DELETE /api/users/{id}` - 删除用户

## 安全特性

1. **权限验证**：前后端双重验证管理员权限
2. **自我保护**：不能删除自己的账号
3. **最小权限**：至少保留一个管理员账号
4. **角色管理**：可以修改用户角色

## 使用说明

1. 使用管理员账号登录系统
2. 点击左侧菜单"后台管理"进入管理后台
3. 根据需要选择相应的管理功能
4. 可以随时点击"返回用户端"切换回普通用户视图

## 注意事项

- 删除操作需要确认，防止误操作
- 删除分类前需确保没有关联图书
- 删除用户前需确保没有未归还的借阅记录
- 修改用户角色需谨慎，避免误操作
